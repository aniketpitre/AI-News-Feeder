## Color Palette

```
Primary Colors:
  Cyan:     #00FFC2  (Main accent, CTAs, highlights)
  Teal:     #00D9FF  (Secondary accent, gradients)
  Blue:     #0080FF  (Tertiary accent)

Backgrounds:
  Black:    #050505  (Main background)
  Card:     #080808  (Card backgrounds)
  Dark:     #0A0A0A  (Slightly lighter for contrast)

Text:
  White:    #FFFFFF  (Primary text)
  Gray:     #FFFFFF @ 60% (Secondary text)
  Muted:    #FFFFFF @ 40% (Tertiary text)

Borders:
  Light:    #FFFFFF @ 10% (Subtle borders)
  Accent:   #00FFC2 @ 30% (Highlighted borders)
```

## Animations

```
1. Hero Section
   - Blob background: Smooth 7s rotation
   - Text fade-in: 0.8s ease-out
   - Gradient text: Pulse effect every 3s
   - Grid background: Subtle fade in

2. Featured Article
   - Fade in on scroll
   - Background orbs animate on hover
   - Border glows with cyan on hover
   - External link icon slides on hover

3. Article Cards
   - Fade in + slide up on scroll (staggered)
   - Card background gradient on hover
   - Text color shifts on hover (white → cyan)
   - Icon animates on hover

4. Navigation
   - Backdrop blur effect
   - Status indicator pulsing
   - Subscribe button glows on hover

5. Bottom Ticker
   - Continuous marquee scroll
   - Gradient background from cyan to blue
```

## Typography

```
Hero Title:
  Font: Black (900 weight)
  Size: 5xl (Desktop) → 8xl (Large screens)
  Line Height: 0.9 (Tight)
  Letter Spacing: Tighter (Compact)

Article Title:
  Font: Bold (700 weight)
  Size: 1.125rem (1.5em)
  Line Height: 1.2
  Max Width: 2 lines

Category Badge:
  Font: Mono (monospace)
  Size: 0.625rem (10px)
  Weight: Bold
  Letter Spacing: Wider

Navigation:
  Font: Medium (500 weight)
  Size: 0.875rem (14px)
  Letter Spacing: Widest (uppercase)
```

## Responsive Breakpoints

```
Mobile (< 640px):
  - 1-column article grid
  - Full-width featured article
  - Stacked navigation
  - Smaller font sizes
  - Touch-friendly buttons (48px min)

Tablet (640px - 1024px):
  - 2-column article grid
  - Horizontal navigation
  - Adjusted spacing

Desktop (> 1024px):
  - 3-column article grid
  - Full navigation with categories
  - Maximum width: 1280px
  - Optimized spacing and padding
```

## Interaction Patterns

```
Hover States:
  ✨ Card: Background gradient + border glow
  ✨ Button: Shadow + scale transform
  ✨ Link: Color change + underline
  ✨ Badge: Background highlight

Active States:
  ✨ Selected category: Cyan background + white text
  ✨ Featured article: Glowing border
  ✨ Navigation link: Cyan underline

Loading States:
  ✨ Articles: Skeleton loaders with pulse
  ✨ Sync: Animated status indicator
  ✨ Search: Debounced with spinner

Error States:
  ✨ Toast notifications (top-right)
  ✨ Red accent color for errors
  ✨ Clear error messages
```
